Files & Processing

The list describes the purpose of files and provides a small survey how to use these files for converting documents:

  • odtreference.odt: File that is used as layout template for ODT.
  • authors.txt: List of Authors of the document.
  • title.wiki: Title of the document
  • shorttitle.wiki: Short title of the document
  • keywords.wiki: Keywords for the document

Commands for WikiMedia to ODT

The following command converts a wikimedia file to ODT

pandoc -f mediawiki -S --reference-odt odtreference.odt -o outfile.odt infile.wiki
